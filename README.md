# Carbon Capture & Storage (CCS) Research Initiative

A portfolio repository documenting the full project management lifecycle of a 
pilot-scale Carbon Capture & Storage (CCS) research initiative, managed using 
a **hybrid Waterfall-Agile (Scrum)** approach.

---

## Objective

Research, design, and validate a pilot-scale CCS system using amine-based 
chemical absorption — targeting ≥90% CO₂ capture efficiency — to inform a 
future commercial-scale deployment decision.
